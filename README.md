## 📖 Semana 1: Fundamentos de JavaScript

### 🎯 Taller de Lógica - Clasificador de Triángulos
**Archivo:** `Semana 1/TallerLógica/clas.triangulos.js`

Un programa interactivo que:
- Solicita al usuario las longitudes de tres lados
- Valida que los lados sean positivos
- Verifica si forman un triángulo válido (desigualdad triangular)
- Clasifica el triángulo como: **Equilátero**, **Isósceles** o **Escaleno**

**Conceptos aprendidos:**
- Variables y entrada de datos con `prompt()`
- Validaciones con condicionales (`if`, `else if`, `else`)
- Funciones personalizadas
- Operadores lógicos y matemáticos
- Manejo de errores y mensajes informativos

### 🎯 Taller Práctico - Fundamentos de JavaScript
**Carpeta:** `Semana 1/TallerPráctico.js_Susana Gutiérrez Callejas/`

Una serie de 8 ejercicios progresivos que cubren:

#### Parte 1: Introducción a JavaScript
- Características del lenguaje
- Diferencias entre lenguajes interpretados y compilados
- Concepto de eventos en programación

#### Parte 2: Variables y Tipos de Datos
- Declaración de variables (`let`, `const`)
- Tipos de datos: números, strings, booleanos
- Diferencias entre `undefined` y `null`

#### Parte 3: Entrada y Salida de Datos
- `prompt()` para entrada de usuario
- `alert()` para mostrar mensajes
- `confirm()` para confirmaciones

#### Parte 4: Operadores
- Operadores aritméticos (+, -, *, /, %)
- Concatenación de strings
- Operadores de comparación (==, ===)
- Operadores lógicos (&&, ||, !)

#### Parte 5: Condicionales
- Estructuras `if`, `else if`, `else`
- Operador ternario
- Validaciones de datos

#### Parte 6: Console Methods
- `console.log()`, `console.info()`, `console.warn()`, `console.error()`
- `console.group()` y `console.groupEnd()`
- `console.time()` y `console.timeEnd()`

#### Parte 7: Comentarios
- Comentarios de una línea (`//`)
- Comentarios de múltiples líneas (`/* */`)
- Buenas prácticas de documentación

#### Parte 8: Proyecto Final
- Integración de todos los conceptos aprendidos
- Formateo de strings con template literals

## 🌐 Semana 2: Manipulación del DOM

### 🎯 Taller DOM - Interactividad Web
**Carpeta:** `Semana 2/Taller DOM/`

Tres partes progresivas que enseñan a crear páginas web interactivas:

#### Parte 1: Arrays orientados al DOM
- Creación y manipulación de arrays
- Métodos `forEach()` y `map()`

- Generación de listas y elementos HTML desde JavaScript

#### Parte 2: Objetos pensados para mostrarse
- Creación y manipulación de objetos
- Acceso a propiedades de objetos
- Creación de tarjetas de información
- Formateo de datos para presentación web

#### Parte 3: Eventos y Interactividad
- Manejo de eventos del DOM
- Creación de elementos dinámicos
- Interacción con formularios
- Validaciones en tiempo real

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura de las páginas web
- **CSS3**: Estilos básicos para presentación
- **JavaScript (ES6+)**: Programación del lado del cliente
- **DOM API**: Manipulación dinámica de páginas web

## 🚀 Cómo ejecutar los proyectos

### Para los ejercicios de consola:
1. Abre el archivo `.js` en tu editor
2. Ejecuta el código en la consola del navegador o con Node.js
3. Sigue las instrucciones en pantalla

### Para los proyectos web:
1. Abre el archivo `index.html` en tu navegador
2. Abre las herramientas de desarrollador (F12)
3. Ve a la pestaña "Console" para ver los resultados
